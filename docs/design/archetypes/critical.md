# 暴击流派：刺客与方位破绽

> 状态：破绽规则与首批 5 件遗物设计稿 · 2026-07-22 · 刺客弹珠 M1 核心循环已实现 2026-07-23
>
> 本文只讨论玩法与美术规格，不给出代码、节点、资源路径或实施计划。Lv1～Lv3 数值均为待试玩锚点；觉醒不一定要新增机制，但应尽量独特，避免沦为纯数值堆叠。

## 1. 定位与幻想

暴击流派的主体是**刺客弹珠**。它能洞察敌人的方位破绽，玩家需要规划弹道，从正确一侧切入敌人轮廓，以一次明确的方位命中造成暴击。

这不是积累层数后自动生效的“校准”，也不是连续命中驱动的狂热连击。它的操作乐趣来自：看见破绽、寻找绕击路线、从正确方向接触目标。

## 2. 印象与关键词

**流派印象**：冷静刺杀、观察站位、绕到侧面、抓住暴露方向、一次干净的致命命中。

**关键词**：**刺客、破绽、方位、洞察、绕击、切入、致命一击**。

赌徒不作为独立流派，只保留为遗物“福尔图娜的骰子”。狂热连击方向已弃用。

## 3. 全新敌方状态：破绽

### 3.1 核心规则

1. 敌人具有上、右、下、左四个方位。
2. 只要刺客弹珠实际在场，普通敌人和 Boss 都显示破绽；刺客弹珠离场后，破绽隐藏。
3. 基础状态下，每个敌人从四个方位中随机显示一个破绽。
4. 只有刺客弹珠能触发破绽；其他来源从正确方向命中也不算破绽暴击。
5. 刺客弹珠从破绽对应方向直接接触敌人时，本次命中造成 **150% 总伤害，即 ×1.5 暴击**。
6. 从非破绽方向接触时只造成普通伤害，现有破绽保留。
7. 命中破绽后，被命中的破绽短暂消失，再随机换到其他合法方向。
8. 刺客弹珠觉醒后，每个敌人额外显示一个破绽，即两个不同方位。

### 3.2 方位判定

方位以屏幕空间中的上、右、下、左为准。判定使用“敌人中心指向实际接触点”的方向角，而不是弹珠飞行方向、发射起点或视觉拖尾方向。

每个方位以正上、正右、正下、正左为中心，允许 **±15°** 的角度容差，单个破绽的总有效角为 **30°**。超出区间即使视觉上接近，也按普通命中处理。

### 3.3 觉醒双破绽刷新

两个破绽必须位于不同方位。命中其中一个时，另一个原位保留，仅被命中的破绽进入短暂空窗并换边。

基础破绽与棱镜破绽共享同一个四方位占用规则：同一方位任何时候最多存在一个破绽。换边时必须避开所有仍存在的基础/棱镜破绽，也不立即回到刚被命中的方向；若当前没有合法方位，则延迟到方位空出后再出现，绝不重叠生成。

一次物理接触最多命中一个破绽、产生一次暴击和一次“命中破绽”结果。结算顺序固定为：确认并消费被命中的破绽 → 结算一次暴击 → 若命中的是基础破绽，则生成/刷新棱镜破绽 → 安排被命中的基础破绽换边。

觉醒刺客未装备千面棱镜时，命中一个基础破绽后，刷新方向从“未占用且不是刚命中方向”的两个方位中随机选择。当前装配规则不允许重复持有同类弹珠，无需叠加多颗刺客的破绽数量。

## 4. 核心循环

```text
刺客弹珠在场 → 敌人显示随机方位破绽
→ 玩家观察方向并规划绕击弹道
→ 从正确一侧直接接触 → 本次命中 ×1.5 暴击
→ 被命中的破绽破碎，短暂空窗后换到其他方向
```

普通命中仍有价值，但流派上限来自持续改变切入方向。回响关心反弹次数和球体成长；刺客关心反弹最终把球送到敌人的哪一侧。

## 5. 首批遗物池总览

| 稀有级别 | id / 名称 | 角色 | 是否能制造暴击 |
|---|---|---|---|
| Common | `assassins_whetstone` 刺客磨刀石 | 扩大方位容差，奖励完美切入 | 否 |
| Uncommon | `fortuna_dice` 福尔图娜的骰子 | 全直伤波动与厄运反转 | 否 |
| Rare | `many_faced_prism` 千面棱镜 | 制造临时额外破绽 | 否 |
| Rare | `scarlet_thread` 猩红丝线 | 把已发生的暴击转为群体收益 | 否 |
| Boss | `execution_decree` 处刑敕令 | 周期性把非暴击直伤改为暴击 | **是** |

普通遗物只能改善破绽、消费暴击或修改伤害，不能把普通命中判为暴击。首批只有 Boss 级“处刑敕令”能在没有命中破绽时制造暴击。

### 5.1 投放资格

资格只用于避免死选，不改变遗物在战斗中的实际效果。

## 6. 刺客磨刀石 `assassins_whetstone`

### 6.1 定位与效果

低门槛路线支持件。它扩大破绽的有效角，但不改变破绽方向、数量或刷新规则。

| 阶段 | 效果 |
|---|---|
| Lv1 | 破绽容差由基础 **±15°** 扩大为 **±18°** |
| Lv2 | 扩大为 **±21°** |
| Lv3 | 扩大为 **±24°** |
| 觉醒 | 新增“完美切入”：接触角位于破绽中心 **±5°** 内时，暴击倍率由 ×1.5 提高为 **×1.75** |

觉醒只提高高精度命中的回报，不会让非破绽命中暴击。若同时拥有福尔图娜，完美切入的组合倍率范围为 **×1.225～×2.625**。

### 6.2 表现

- 遗物外观：一块狭长黑色磨刀石，表面嵌着四条银色角度刻线；觉醒后中央多一条金线。
- 图标：刀尖指向一段扇形刻度，外圈表示当前容差，中心细金区表示完美切入。
- 场上反馈：装备后，破绽短弧略微加宽；觉醒后的中心 ±5° 使用更细、更亮的金色内芯。
- 触发特效：普通破绽暴击保持银白切痕；完美切入增加一次极短的金线贯穿和更清脆的高音刀鸣。
- 数字反馈：完美切入显示独立短标签“完美”，避免仅靠更大的伤害数字判断。

### 6.3 边界

- 多个角度扩展来源只改变容差，不增加破绽数量。
- 完美切入仍是一次破绽暴击，可触发消费暴击的遗物。
- Boss 级遗物制造的普通暴击没有方位中心，因此不能触发完美切入。

## 7. 福尔图娜的骰子 `fortuna_dice`

### 7.1 直伤定义

每个实际结算的即时、非持续伤害实例独立掷一次六面骰。六个等概率骰面暂定为：

```text
×0.7 / ×0.8 / ×0.9 / ×1.1 / ×1.3 / ×1.5
```

倍率范围为 **×0.7～×1.5**，Lv1 的长期平均倍率为 **×1.05**。适用范围包括：

- 弹珠接触伤害；
- 技能直伤；
- 即时爆炸与溅射；
- 即时连锁伤害；
- 其他即时次级伤害，例如猩红丝线。

DOT tick 与持续区域伤害不适用。一个 AOE 同时伤害三个目标时，三个目标分别掷骰。

### 7.2 等级与觉醒

| 阶段 | 效果 |
|---|---|
| Lv1 | 每次适用直伤独立掷一次六面倍率骰：**×0.7/×0.8/×0.9/×1.1/×1.3/×1.5** |
| Lv2 | 首次掷出最低面 **×0.7** 时重掷一次，并采用第二次结果 |
| Lv3 | 首次掷出任意低倍率面（×0.7/×0.8/×0.9）时重掷一次，并取两次中较高的结果 |
| 觉醒 | “命运反转”：取消重掷；低倍率面直接翻到对应高倍率面：×0.7→×1.5、×0.8→×1.3、×0.9→×1.1 |

Lv2 的第二次结果即为最终结果，即使仍是 ×0.7；Lv3 才会在重掷后保留较高值。觉醒后每个伤害实例独立完成翻面，不保存跨实例状态，因此多目标伤害不会争抢重掷或幸运结果。

### 7.3 与暴击的关系

骰子波动不是暴击，也不触发暴击事件。骰子倍率与破绽/处刑敕令的暴击倍率乘算：

```text
普通破绽暴击：0.7 × 1.5 = 1.05 ～ 1.5 × 1.5 = 2.25
完美切入暴击：0.7 × 1.75 = 1.225 ～ 1.5 × 1.75 = 2.625
```

### 7.4 表现

- 遗物外观：象牙色六面骰，六个面分别嵌入冷蓝到暖金的渐变刻度，而不是显示固定倍率数字。
- 图标：倾斜旋转的骰子，背后并列一条向下冷色刻度和向上金色刻度。
- 每次直伤：伤害数字旁短暂出现小骰面与实际倍率；低于 ×1.0 向下轻坠，高于 ×1.0 向上弹起。
- Lv2/Lv3 重掷：第一枚骰子短暂弹起，第二枚落下；Lv3 最终采用的较高骰面留下银色框。
- 觉醒翻面：低倍率骰面沿水平轴翻转为对应高倍率面，并留下短促金色轨迹。
- 音效：低倍率为轻微木质落桌声，高倍率为清脆骨骰声；重掷使用连续两次落桌声，命运反转使用一次翻面锁定声。

### 7.5 边界

- 不重复修改同一个伤害实例。
- 每个伤害实例独立掷骰、重掷或翻面，不保存跨实例状态。
- 猩红丝线等即时次级伤害会独立掷骰，但不会继承触发它的原始骰子结果。

## 8. 千面棱镜 `many_faced_prism`

### 8.1 定位与效果

高阶破绽供给件，允许遗物增加第三个破绽，但 Lv1～Lv3 只生成**临时额外破绽**，避免直接取代刺客觉醒。

成功命中一个基础破绽后，在该敌人的未占用方位立即生成一个“棱镜破绽”。它与基础破绽一样可被刺客触发并造成 ×1.5 暴击，但颜色和生命周期不同。

| 阶段 | 效果 |
|---|---|
| Lv1 | 棱镜破绽持续 **2 秒** |
| Lv2 | 持续 **3 秒** |
| Lv3 | 持续 **4 秒** |
| 觉醒 | 棱镜破绽不再按时间消失，直到被命中、敌人离场或刺客弹珠离场 |

基础刺客最多形成“1 个基础 + 1 个棱镜”破绽；觉醒刺客最多形成“2 个基础 + 1 个棱镜”破绽，即允许第三个破绽存在。

### 8.2 刷新与递归边界

- 只有命中**基础破绽**才能生成棱镜破绽。
- 命中棱镜破绽可以暴击并触发暴击收益，但不会再生成新的棱镜破绽。
- 同一敌人同时最多存在一个棱镜破绽；已有时再次命中基础破绽只刷新持续时间，不重掷方向。
- 基础破绽与棱镜破绽共享四方位占用；任何生成或换边都不能与另一种破绽重叠，也不能使用刚被命中的方向。
- 一次接触即使位于多种视觉范围的交界，也只消费一个破绽并产生一次暴击；方位重叠不作为合法状态存在。

### 8.3 表现

- 遗物外观：不规则透明棱镜，内部折射出四个错位的刀口倒影；觉醒后中心出现稳定紫金光核。
- 图标：三角棱镜把一条白色刀光折成三条，其中一条使用紫色虚线表示临时破绽。
- 棱镜破绽：保持基础破绽的形状语法，但采用紫白双层轮廓，并带随剩余时间收缩的透明外环。
- 生成特效：基础破绽碎裂时，一片光屑折向新方位并展开棱镜破绽。
- 觉醒表现：倒计时外环消失，紫白轮廓变为稳定实线，表示不再自然过期。
- 音效：生成时使用短促玻璃折射声；命中时在普通暴击刀鸣后叠加一声细碎晶响。

## 9. 猩红丝线 `scarlet_thread`

### 9.1 定位与效果

暴击群体 payoff。它不制造暴击，只消费已经发生的破绽暴击或 Boss 遗物暴击，把单体精确命中转成有限的群体直伤。

每次暴击后，从受击目标向最近的另一名可攻击敌人拉出一条猩红丝线并结算一次切割。切割基础值取自触发命中在**骰子与暴击乘区之前**的直伤值，然后作为新的即时直伤独立接受福尔图娜倍率。

| 阶段 | 效果 |
|---|---|
| Lv1 | 丝线对最近 1 名其他敌人造成触发基础直伤的 **25%** |
| Lv2 | 提高为 **35%** |
| Lv3 | 提高为 **45%** |
| 觉醒 | 规则改为同时连接最近的 **2 名不同敌人**，每条仍造成 45% |

### 9.2 边界

- 丝线切割是即时直伤，会独立触发福尔图娜。
- 丝线切割**不能暴击**，不能命中破绽，也不能触发新的猩红丝线。
- 每次原始暴击最多触发一次猩红丝线；觉醒只是把一次触发分成两个目标。
- 场上没有其他敌人时不触发，也不储存到下一次。
- 同一条觉醒丝线不会两次选择同一目标。

### 9.3 表现

- 遗物外观：缠绕在银色线轴上的暗红丝线，线轴两端是微型匕首；觉醒后分裂为双线轴。
- 图标：一个被红线贯穿的破绽符号，线末端连接另一道敌人剪影。
- 触发特效：暴击切痕结束后，细红线瞬间拉向目标，绷紧一拍再断裂；伤害在断裂瞬间出现。
- 觉醒表现：红线在源目标处分叉成清晰的 Y 形，两条线错开极短时间结算以保证可读性。
- 音效：先是丝线绷紧声，再接一声很轻的割裂声；不使用爆炸或雷链音色。
- 数字反馈：次级伤害数字带“丝线”短标签，不使用暴击样式。

## 10. 处刑敕令 `execution_decree`

### 10.1 定位与效果

Boss 级遗物，也是首批遗物中唯一能在没有命中破绽时制造暴击的内容。它把周期明确显示给玩家，不使用隐藏暴击概率。

每个原本不会暴击的直伤实例为敕令增加 1 格。计数满后进入“已武装”，下一次独立直伤事件的主目标被改为 **×1.5 暴击**，随后计数归零。弹珠、技能、即时爆炸、溅射和连锁均可推进计数。

该进度没有时间窗，不会因未命中、停顿、换目标或切换伤害来源而中断；它是跨事件累计的处刑进度，不是狂热连击。

| 阶段 | 效果 |
|---|---|
| Lv1 | 累积 **7** 次非暴击直伤后武装，下一次主目标直伤暴击；即单体连续命中时每第 **8** 次暴击 |
| Lv2 | 累积 **6** 次后武装；单体连续命中时每第 **7** 次暴击 |
| Lv3 | 累积 **5** 次后武装；单体连续命中时每第 **6** 次暴击 |
| 觉醒 | “血书加速”：每次刺客命中破绽时，额外为下一轮敕令增加 **2 格**；该次破绽暴击本身不消费敕令 |

### 10.2 暴击与递归边界

- 处刑敕令制造的暴击可触发猩红丝线等“暴击后”收益。
- 处刑暴击与福尔图娜乘算，普通组合范围为 ×1.05～×2.25。
- 已经因破绽而暴击的直伤不被再次改写为第二次暴击，基础状态也不推进普通敕令计数。
- 觉醒后的破绽暴击只增加 2 格，不会在同一次命中上立刻制造第二次暴击。
- 若普通进度或觉醒的 +2 格达到/越过阈值，敕令在当前独立直伤事件全部结算后进入“已武装”：进度封顶、不保留溢出。
- 已武装后，下一次独立直伤事件的主目标若原本不会暴击，则改为暴击并归零；该事件的其他目标不被自动改为暴击。
- 已武装期间再次命中破绽不继续积累，也不储存第二次处刑暴击。
- 处刑暴击本身不计入下一轮；同一事件的其他非暴击直伤在事件结束后可开始积累新一轮进度。
- 由暴击 payoff 产生的即时次级直伤，例如猩红丝线，不推进敕令，防止自循环。

“主目标”采用玩家可见语义：弹珠/投射物为直接接触目标，连锁为第一个跳跃目标，爆炸或无明确主目标的溅射为距离中心最近的目标。

### 10.3 表现

- 遗物外观：黑色处刑卷轴，以暗红蜡封封口；蜡封外围有与当前等级阈值一致的格点。
- 图标：展开的敕令、断头刀形印章和环形计数格，Boss 边框使用黑金而非普通稀有度颜色。
- 常驻反馈：HUD 显示清晰的敕令格数；每次推进亮一格，最后一格武装时蜡封完整闭合。
- 触发特效：被改写为暴击的目标上方短暂落下一枚暗红印章，命中瞬间印章被一道黑金刀线斩开。
- 觉醒表现：刺客命中破绽时，两滴红墨飞向计数器并点亮空余格；若只剩一格，第二滴融入闭合蜡封。武装只在当前直伤事件结束后闭合，避免同一爆炸中途突然改写目标。
- 音效：推进为低沉印章声；武装为卷轴拉紧声；处刑暴击为印章碎裂与短促刀落声。

## 11. 五件遗物的组合关系

```text
刺客磨刀石 ──提高破绽命中率 / 完美切入倍率
千面棱镜   ──增加可操作的临时破绽
                     ↓
                破绽暴击 ×1.5
                     ↓
猩红丝线     ──把已发生的暴击转成群体直伤

福尔图娜的骰子 ──修改所有即时直伤；与暴击乘算
处刑敕令       ──Boss 级周期制造暴击，并接入猩红丝线
```

### 11.1 多目标直伤事件边界

同一次爆炸、溅射或连锁视为一个“独立直伤事件”，其中每个实际受伤目标仍是一个独立伤害实例：

- 福尔图娜由每个实例自行掷骰/重掷/翻面，不共享跨实例状态，因此不依赖目标结算顺序。
- 处刑敕令统计该事件内全部合格的非暴击实例，但只在整个事件结束后更新为“已武装”，不会在一次 AOE 中途突然让某个兄弟目标暴击。
- 已武装的处刑敕令只改写下一事件的主目标；其余目标按普通规则结算，并在事件结束后开始积累下一轮进度。
- 猩红丝线等由暴击产生的次级直伤，在父事件完成后结算；它独立受福尔图娜影响，但不推进处刑敕令。

该规则保证每目标独立抽取，同时避免同一多目标事件中由不可见遍历顺序决定谁获得处刑暴击。

建议满配形态：刺客弹珠 + 磨刀石 + 千面棱镜构成操作核心；猩红丝线提供群体兑现；福尔图娜提供通用波动；处刑敕令作为 Boss 级替换位或超高价值第五件。五槽限制下，玩家需要在稳定破绽、群体收益、随机波动和周期暴击之间选择，并非五件全部无脑必拿。

## 12. 共用视觉语言

- 主色：冷白、银灰、低饱和青；破绽暴击瞬间加入金色针芒。
- 形状：单侧刀口、四向刻度、细切线、缺角菱形；避免火的扩散圆、闪电折线和爆破冲击环。
- 破绽依附敌人轮廓，不阻挡弹珠，也不接受伤害。
- 基础破绽使用银白实线；棱镜破绽使用紫白双层；完美切入区域使用极细金线。
- 暴击数字显示固定暴击标签；福尔图娜倍率、丝线伤害和敕令计数各有独立样式，不能全部复用暴击数字。

## 13. 平衡与验证边界

- 只有刺客命中破绽与 Boss 级处刑敕令能产生暴击事件。
- 普通暴击倍率固定 ×1.5；磨刀石觉醒的完美切入为 ×1.75。
- 福尔图娜只修改即时直伤，DOT 与持续区域不参与。
- 千面棱镜最多增加一个额外破绽；觉醒刺客配棱镜最多三个，不覆盖全部四向。
- 猩红丝线和其他暴击 payoff 不能暴击、不能自触发，也不能推进处刑敕令。
- 处刑敕令以显式计数制造暴击，不引入隐藏随机暴击率。
- 护甲尚未接入主线，本批遗物不规划穿甲、削甲或护甲桥梁。
- 首批五件全部服务暴击核心或作为通用直伤件，跨流派桥梁留到破绽玩法验证后再设计。
