# 暴击流派设计：精确爆发（待确认）

> 状态：印象、关键词与核心 Buff 决策稿 · 2026-07-22
>
> 本文只讨论玩法与美术规格，不给出代码、节点、资源路径或实施计划。所有名称与数值均未定案。

## 1. 先确认流派给人的印象

### 推荐：精确猎手

暴击应像一次经过准备的精确命中：玩家先观察板面、选择目标、积累准备度，再把一记清晰而克制的重击打进弱点。它的兴奋点不是“骰中了”，而是“这条路线是我算出来的”。

**推荐印象**：冷静、锐利、克制、危险、瞬间爆发、命中前安静、命中后干净利落。

**推荐关键词**：

- 核心词：**校准、弱点、锁定、蓄势、兑现**。
- 风险词：**失准、断线、过量**。
- 视觉词：**准星、刻度、切线、针芒、白金闪光**。

### 两个备选方向

| 方向 | 印象与关键词 | 优点 | 主要问题 |
|---|---|---|---|
| 赌徒 / Jackpot | 幸运、翻倍、孤注一掷、连爆 | 即时刺激强，遗物容易理解 | 物理轨迹已经有随机性，再叠概率会削弱玩家归因 |
| 狂热 / 连击 | 节奏、不断命中、升温、暴走 | 爽快，适合多段命中 | 容易与火的高节奏、回响的长轨迹重叠 |

**当前推荐结论**：采用“精确猎手”，明确反对把暴击做成单纯堆叠百分比的通用伤害流派。

## 2. 核心循环

```text
选择首要目标 → 用主球完成可读的精确动作 → 获得「校准」
→ 校准达到阈值，进入“已对准” → 下一次合格命中确定暴击
→ 选择立刻兑现，或继续承担失准风险换取过量收益
```

主球身份在本流派中很重要：允许某颗暴击弹珠担任主球后，玩家从瞄准阶段就能确认“这一发由谁执行精确命中”。跟随球、溅射、DOT 与自动追踪实体不应轻易替代主球完成核心兑现。

## 3. 全新 Buff：校准

### 3.1 设计语义

`校准`是附着于当前主球/本发的正面 Buff，用分段准星表示准备度。它本身不按层直接加伤；层数达到阈值后，将下一次符合条件的主球直击转为**确定暴击**。

这样做的目的：

- 把暴击从后台概率改成玩家能计划的资源。
- 让 Buff 有明确的“积累—武装—消耗”三态，而不是另一种常驻倍率。
- 为遗物提供保留、转移、过量、失准保护等机制空间。
- 让暴击动画可以在命中前预告，不必等伤害数字出现才知道发生了什么。

### 3.2 暂定边界

- 只有**直接主球命中**可消耗已武装的校准；溅射、DOT、区域、连锁和召唤实体默认不能代为消耗。
- 暴击发生时应给出确定性的视听反馈，不与普通高伤共用同一表现。
- 校准达到阈值后不继续线性提高暴击率；溢出资源若保留，应转成另一种可读收益。
- 不允许暴击产物无条件再次暴击，避免递归爆发。
- 换主球、落球与一发结束时如何处理校准，必须在定案时明确。

### 3.3 校准如何获得：三个候选

| 候选 | 规则印象 | 判断 |
|---|---|---|
| A 首次接触 | 主球依次命中本发尚未命中的不同敌人，逐步校准；重复命中同一目标不增加 | 值得优先验证路线选择，但还需增加“精确动作”门槛，避免杂兵房被动填满 |
| B 干净反弹 | 未碰敌人时完成有效墙面/挡板反弹，逐步校准 | 视觉直观，但会与回响争夺机制身份 |
| C 连续命中 | 在限定节奏内连续命中便校准，空窗则失准 | 爽快，但容易滑向狂热连击流 |

三者目前都只是机制候选，须先确认“精确猎手”的流派气质再选。若气质获确认，可优先细化 A，但 Boss 或场上只有单一敌人时需要“命中不同弱点区域”等补足规则，且杂兵房必须防止校准被动填满。

### 3.4 仍需你确认的 Buff 规则

1. 达到阈值后，是下一次合格命中立即强制暴击，还是允许玩家选择继续蓄势？
2. 暴击后清空全部校准，还是只消耗阈值所需层数？
3. 校准能否跨发保留？当前推荐**不跨发**，以保持每发的瞄准节奏。
4. 失准是“重复命中不涨层”，还是会损失已有层数？当前推荐前者，减少挫败。
5. 暴击倍率固定，还是由溢出校准提高？当前推荐固定基础倍率，溢出由遗物改变用途。

## 4. 弹珠与遗物协同骨架（Buff 定案后再命名）

以下只锁定内容角色，不锁定具体物品：

| 角色 | 设计职责 | 边界 |
|---|---|---|
| 暴击弹珠 / enable | 可担任主球；让“首次接触”更容易积累校准，并清楚显示当前执行者 | 不能仅写成“暴击率更高” |
| 校准遗物 / core | 建立校准的获得、武装与消耗循环 | 单独拿到时也必须能运作 |
| 路线支持 / support | 提供一次失准保护、改变目标顺序价值或保留部分校准 | 不替玩家自动完成整条路线 |
| 弱点支持 / support | 给优先目标可读弱点，使单体/Boss 场景也能积累 | 弱点不能成为随机小像素 |
| 过量兑现 / payoff | 已武装后继续完成高难动作，把溢出校准转为额外效果 | 必须有封顶，不能无限乘区 |
| 桥梁遗物 | 将校准或暴击结果连接到护甲、冰冻碎裂等可信机制 | 在护甲语义确定前不锁定 |

Buff 规则确认后，下一轮再给出正式的暴击弹珠、核心遗物、支持件、payoff 与桥梁件名称和完整机制。

## 5. 视觉语言

### 5.1 色彩与形状

- 主色：冷白、银灰、低饱和青；暴击瞬间加入极短的金色针芒。
- 形状：同心准星、四向刻度、细直切线、菱形弱点；避免火的扩散圆、闪电的折线与爆破的厚重冲击环。
- 动态：蓄势阶段收拢、变细、变稳；兑现阶段只闪一次，随后迅速熄灭。

### 5.2 弹珠表现

- 暴击主球表面带可读的旋转刻度环；校准每增加一段，刻度环便少量对齐。
- “已对准”时刻度停止漂移，球心出现一根稳定针芒，不持续大幅发光。
- 失准不使用爆炸或红叉；采用准星轻微错位与短促金属走音，传达“未增长”而非“惩罚失败”。

## 6. 图标方向

| 对象 | 图标构图 | 识别重点 |
|---|---|---|
| 暴击流派 | 被一根针穿过的同心准星 | 小尺寸下仍能看清中心命中 |
| 校准 Buff | 分为若干段的圆形刻度环 | 层数通过亮起段数读取 |
| 已对准 | 完全闭合的准星 + 中央菱形 | 与普通校准状态明显区分 |
| 失准保护 | 略偏移但未断裂的双准星 | 表示容错，不用盾牌图形 |
| 过量收益 | 准星外多一圈细刻度 | 表示阈值之外的积累 |

图标避免骰子、四叶草和扑克牌符号，除非最终改选“赌徒”方向。

## 7. 特效与实体需求

### 7.1 校准获得

- 合格命中后，从目标弱点向主球回收一条极细光线，准星补亮一段。
- 同一时刻只强调一次获得，避免多目标事件把屏幕变成光线网。
- 音效按层数逐步升高，但变化应克制；达到阈值时给独立的闭锁声。

### 7.2 暴击兑现

- 命中点先出现 1 帧左右的细白十字，再出现短促金色切痕与较大的暴击数字。
- 不生成持续火花雨；余光沿入射方向穿过目标少量距离，强调“精准穿点”。
- 若触发过量效果，增加第二圈刻度扩散，而不是把基础暴击做得更炸。

### 7.3 弱点实体候选（仅在方向确认后引入）

- 弱点依附敌人，可随敌人移动；始终处于角色轮廓内或轮廓边缘，不漂在无关位置。
- 同一普通敌人默认仅显示一个主要弱点；Boss 可有多个分区弱点，但同屏高亮数量必须受限。
- 弱点被命中后应明确进入“已命中/冷却”状态，不能只靠颜色微差。
- 弱点不遮挡敌人状态图标、血条或其他流派的重要反馈。

本流派暂不需要独立溅射物。若遗物产生“穿刺线”或“碎片”，它们属于暴击结果的次级实体，默认不能再次积累或消耗校准。

## 8. 反馈与可读性

- 玩家在发射前应能知道当前校准层数、阈值和本次可否暴击。
- 暴击数字需要固定版式与短标签，避免仅靠数字大小判断。
- 被弱点选中的目标要与“带电”“冻结”“感染宿主”保持轮廓语言差异。
- 校准来源、失准原因和消耗对象都必须能在短时间内回看或理解。

## 9. 平衡边界

- 暴击倍率属于独立乘区，但同类倍率只在该区内合并，避免多件遗物指数相乘。
- 暴击应偏爱少量直接大击，但不能让 DOT、多段与溅射完全失去构筑价值；桥梁可转化结果，不能让所有子伤害自由暴击。
- 单体 Boss 必须有校准路径；杂兵多的房间则要防止首次接触过于容易。
- 已对准状态必须有明确上限；过量收益有硬封顶或每发次数上限。
- 护甲若继续采用固定减伤，会让暴击天然获得额外优势，因此护甲语义必须与暴击一起评估。

## 10. 与其他流派的候选桥梁

- 暴击 ↔ 冰：命中冻结目标的弱点造成“精准碎裂”，主题可信，但不能绕过生命值。
- 暴击 ↔ 毒：对宿主的精确处决强化传播，而不是让每个毒 tick 暴击。
- 暴击 ↔ 护甲/物理：校准命中可形成破甲或穿甲，但须等护甲主线语义确认。
- 暴击 ↔ 回响：高难反弹可提供一次校准保护，但不直接按反弹次数重复加层，以免身份重叠。

以上均为候选，不进入总桥梁图。
