# 暴击流派：刺客与方位破绽

> 状态：破绽核心规则与双破绽刷新均已确认 · 2026-07-22
>
> 本文只讨论玩法与美术规格，不给出代码、节点、资源路径或实施计划。

## 1. 定位与幻想

暴击流派的主体是**刺客弹珠**。它能洞察敌人的方位破绽，玩家需要规划弹道，从正确一侧切入敌人轮廓，以一次明确的方位命中造成暴击。

这不是积累层数后自动生效的“校准”，也不是连续命中驱动的狂热连击。它的操作乐趣来自：看见破绽、寻找绕击路线、从正确方向接触目标。

## 2. 印象与关键词

**流派印象**：冷静刺杀、观察站位、绕到侧面、抓住暴露方向、一次干净的致命命中。

**关键词**：**刺客、破绽、方位、洞察、绕击、切入、致命一击**。

狂热连击方向已弃用。赌徒不作为流派方向展开，只保留为单件高波动遗物“福尔图娜的骰子”。

## 3. 全新敌方状态：破绽

此前的「校准」方案已删除。新的状态是附着在敌人身上的**破绽**：它属于 Buff/状态体系中的敌方负面状态或标记，载体、触发方式和玩家操作都与校准完全不同。

### 3.1 已确认的核心规则

1. 敌人具有上、右、下、左四个方位。
2. 只要至少一颗刺客弹珠实际在场，普通敌人和 Boss 都显示破绽；刺客弹珠全部离场后，破绽隐藏。
3. 破绽在四个方位中随机选择一个方向显示，基础状态下每个敌人同时显示一个。
4. 只有刺客弹珠能触发破绽；其他弹珠即使从正确方向接触也只造成普通伤害。
5. 刺客弹珠从破绽对应方向接触敌人时，视为击中破绽。
6. 击中破绽时，**本次直接命中造成 150% 总伤害，即最终暴击倍率为 ×1.5**。
7. 从非破绽方向接触时只造成普通伤害，现有破绽保留。
8. 命中后，被命中的破绽短暂消失，再随机换到其他可用方向。
9. 刺客弹珠觉醒后，每个敌人额外显示一个破绽，即同时显示两个不同方位。

### 3.2 方位命中的设计定义

方位以屏幕空间中的上、右、下、左为准。判定使用“敌人中心指向实际接触点”的方向角，而不是弹珠飞行方向、发射起点、弹珠上一帧所在位置或视觉拖尾方向。

每个方位以正上、正右、正下、正左为中心，允许 **±15°** 的角度容差，因此单个破绽的总有效角为 **30°**。超出该区间即使视觉上接近破绽，也按普通命中处理。

### 3.3 双破绽刷新规则

觉醒状态下，两个破绽必须位于不同方位。命中其中一个时，只让被命中的破绽进入短暂空窗并换边，另一个继续保留。

换边时不能与仍存在的破绽重叠，也不立即回到刚被命中的方向，因此从其余两个空闲方位中随机选择。当前装配规则不允许重复持有同类弹珠，因此无需处理多颗刺客叠加破绽数量。

## 4. 核心循环

```text
刺客弹珠在场 → 敌人显示随机方位破绽
→ 玩家观察方向并规划绕击弹道
→ 从正确一侧直接接触 → 本次命中 ×1.5 暴击
→ 被命中的破绽破碎，短暂空窗后换到其他方向
```

普通命中仍有价值，但流派上限来自持续改变弹珠切入方向。与回响相比，回响关心反弹次数和球体成长；刺客关心反弹最终把球送到敌人的哪一侧。

## 5. 弹珠设计：刺客弹珠

### 5.1 身份

- 是暴击流派的 enable；只要实际在场就显示破绽，并且只有刺客弹珠能触发。
- 觉醒效果：每个敌人从一个破绽增加为两个不同方位的破绽。
- 基础伤害不应再额外高于同级弹珠；强度来自玩家成功命中破绽。
- 跟随球、溅射、DOT、区域伤害和自动追踪实体默认不触发破绽。

### 5.2 玩家决策

- 直取眼前目标，获得稳定普通伤害。
- 利用墙面、敌人和场地结构绕到破绽方向，争取 150% 暴击。
- 当多个敌人的破绽方向不一致时，选择一条能连续切入多个有效侧面的路线。

## 6. 遗物设计：福尔图娜的骰子

**定位**：单件高波动赌徒遗物，不是另一套赌徒流派，也不是刺客弹珠的必需核心。

**效果**：每次直接命中独立从 **×0.7～×1.5** 的区间抽取伤害倍率，即伤害在 **−30%～+50%** 之间波动。

必须明确：

- 每次直接命中重新掷骰，上一击结果不影响下一击。
- ×0.7 表示造成原伤害的 70%，×1.5 表示造成原伤害的 150%。
- 骰子波动本身不叫“暴击”，不触发“击中破绽”类条件。
- DOT tick、持续区域、溅射和连锁分支不属于“直接命中”，不各自重掷骰子。

### 6.1 抽取与反馈

每次直接命中出现一次短促骰面结果；低于 ×1.0 使用偏冷色和下行刻度，高于 ×1.0 使用偏暖色和上行刻度，正好 ×1.0 使用中性反馈。显示不能遮挡破绽方向或暴击数字。

### 6.2 与破绽的关系

骰子倍率与破绽暴击 **×1.5 乘算**。刺客弹珠命中破绽时，组合总倍率范围为：

```text
最低：0.7 × 1.5 = 1.05
最高：1.5 × 1.5 = 2.25
```

因此即使掷出最低倍率，成功命中破绽仍造成 105% 基础伤害；最高可造成 225% 基础伤害。

## 7. 后续遗物角色

除福尔图娜的骰子外，暴击流派仍需要以下遗物角色，名称和机制留待下一轮设计：

| 角色 | 设计空间 | 边界 |
|---|---|---|
| 洞察支持 | 提前显示下一次破绽方向，或减少换边空窗 | 不能自动把破绽转到来球方向 |
| 绕击支持 | 成功反弹后扩大一次破绽判定容差 | 不直接按反弹次数加伤 |
| 致命切入 payoff | 单次破绽暴击满足额外方位条件时，获得处决或穿刺类收益 | 每次独立判断，不按连续次数成长 |
| 方位操控 | 以有限次数重掷不利的破绽方向 | 必须让玩家主动选择，不能全自动追踪 |
| 桥梁遗物 | 将破绽命中连接到碎冰、宿主处决等其他流派结果 | 首批遗物先验证暴击核心，桥梁后置 |

## 8. 视觉语言

### 8.1 破绽方向

- 每个敌人轮廓外侧只显示一个短弧形切口，分别贴在上、右、下、左。
- 破绽形状采用“断开的菱形刀口 + 向内箭尖”，不照搬其他游戏的具体图形。
- 颜色以冷白和低饱和青为主，中心有一条细金线；与冻结蓝块、带电黄弧、感染绿雾保持区别。
- 破绽必须随敌人移动，并始终贴合轮廓对应侧。

### 8.2 刺客弹珠

- 球面使用四瓣对称刀锋纹；在场运动时，其中朝向当前运动方向的一瓣略亮。
- 拖尾是一条短而细的银白切线，不使用烟雾、火星或闪电折线。
- 接近有效破绽侧时，破绽可轻微收紧或变亮，但不能提前替玩家自动确认命中。

## 9. 图标方向

| 对象 | 图标构图 | 识别重点 |
|---|---|---|
| 暴击流派 | 四向框中的单侧刀口 | 一眼看出“只有一个方向有效” |
| 刺客弹珠 | 四瓣刀锋包围球心 | 与普通准星和远程武器区分 |
| 破绽状态 | 缺一角的菱形轮廓 + 内向箭尖 | 小尺寸仍能看出暴露方向 |
| 福尔图娜的骰子 | 倾斜旋转的古典骰子，背后各有一条下坠/上升刻度 | 同时表达负收益与高上限 |

## 10. 特效与反馈

### 10.1 击中破绽

- 接触点先出现极短的白色刀口，随后沿命中侧向敌人中心划过一条金色切线。
- 破绽图形向内闭合并碎成少量几何片，明确表示“正确侧已命中”。
- 暴击数字显示固定的暴击标签与 150% 结果，不只依靠字号变大。
- 音效使用短促金属切割声和低频确认音，不使用爆炸轰鸣。

### 10.2 未击中破绽

- 使用普通命中反馈；破绽保持原方向。
- 不出现红叉或失败文字，避免正常伤害被感知为完全无效。

### 10.3 破绽换边

- 旧破绽命中后先彻底消失；新方向出现时，以贴着敌人轮廓移动的细线完成一次短促转移。
- 同屏多个敌人换边时应错开强调音，避免信息拥挤。

## 11. 实体与状态规则

- 破绽依附敌人，不是独立飞行物，不阻挡弹珠，也不接受伤害。
- 破绽跟随敌人移动，但使用屏幕四向，不随敌人美术朝向旋转。
- 弱点显示层级低于伤害数字，高于敌人主体；不能遮挡血条与关键状态图标。
- 敌人死亡或离场时，其破绽结束；最后一颗刺客弹珠离场时，场上全部破绽隐藏。
- 破绽命中产生的切线只负责反馈，不造成第二次伤害，也不再次触发暴击。

## 12. 平衡边界

- 150% 是**总伤害倍率 ×1.5**，不允许同时被解释为额外 +150%。
- 基础破绽只放大刺客弹珠的直接命中，不自动放大 DOT、溅射、连锁、区域或召唤物伤害。
- 破绽判定区域必须能看清，但不能覆盖敌人整侧的大部分轮廓而变成自然命中。
- 高频反弹困住单个敌人时，需要通过换边空窗或每发触发限制防止原地连续暴击。

## 13. 候选桥梁

- 暴击 ↔ 冰：从破绽侧命中冻结敌人，产生“精准碎裂”。
- 暴击 ↔ 毒：命中宿主破绽时强化死亡传播，但不让毒 tick 暴击。
- 暴击 ↔ 回响：反弹可帮助绕到正确方位，但不按反弹次数直接提高暴击倍率。

这些桥梁暂不进入总桥梁图。

## 14. 已确认边界

- 觉醒后的两个破绽中，命中一个时，另一个原位保留，仅刷新被命中的破绽。
- 被命中的破绽从“未占用且不是刚命中方向”的两个方位中随机换边。
- 护甲尚未接入游戏主线，暴击流派当前不规划穿甲、削甲或护甲桥梁。
