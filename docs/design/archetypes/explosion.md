# 爆破流派：保龄球位移引擎

> 本文是玩法/美术规格草案，不是实现说明。文中全部数值均为待试玩锚点，须以实际对局反馈校准。

## 定位与幻想

爆破的核心不是一团范围伤害，而是把敌人炸成会互相撞击的保龄球。玩家选择引爆时机和敌群密度，让冲量传过人群、撞上墙面、再由殉爆接力；每次爆炸都应改变台面位置关系。

## 印象与关键词

震荡、冲量、击飞、碰撞、保龄球、引信、连锁、余波。它应沉重、有方向、有后果，而不是轻飘的特效雨。

## 核心循环

炸弹弹珠命中制造初始爆炸；火药压机提供延迟炸弹，允许玩家安排“哪次命中引爆”；震荡波把爆炸转成强击飞与敌人碰撞；殉爆在爆炸击杀或撞墙时接过链条。高水平玩法是先聚团，再以正确角度让敌人撞向更多敌人或边界。

## 弹珠与遗物协同

| 角色 | 名称 / id | 玩家可感知作用 |
|---|---|---|
| 起点 | 炸弹弹珠 / `bomb_marble` | 命中后在接触点爆炸。 |
| 节奏 | 火药压机 / `powder_press` | 若干次命中后贴上延迟炸弹，由后续命中引爆。 |
| 位移 | 震荡波 / `concussive` | 增强击飞，使敌人碰撞并传递冲量。 |
| 兑现 | 殉爆 / `sympathetic` | 爆炸击杀或撞墙后引发有限次数的二次爆炸。 |
| 桥梁 | 燃烧弹 / `incendiary`、毒气弹 / `gas_shell` | 分别连接火与毒。 |

雏形应是炸弹弹珠和震荡波，先证明敌人位移与互撞值得操作；殉爆是群体布局正确时的兑现；火药压机提供节奏而不应成为无脑自动增伤。敌人物理、冰封棋子等已有流派机制可以创造聚团机会，但爆炸本身必须来自弹珠体系。

## 视觉语言

主色为煤黑、铁灰、爆焰橙黄和冲击白。爆炸中心应紧实，冲击方向通过地面尘环、碎屑流与被击飞敌人的拖尾表达。延迟炸弹是清晰的金属小包或闪烁引信，而非难以区分的状态图标；撞墙的敌人需留下短暂的凹陷、火花或震纹。

## 图标方向

炸弹弹珠为黑色圆珠与短引信；火药压机为压实火药的机械压板；震荡波为向外推开的三层扇形冲击；殉爆为两枚相邻炸弹相继点亮；燃烧弹为橙红液体的炮弹；毒气弹为带绿烟的炮弹。轮廓应强调“重物、冲量、方向”。

## 特效需求

- 初爆包含短促闪白、压缩火球、随后扩张的尘环三段节奏。
- 被击飞敌人有清楚的方向箭形拖尾，速度越高拖尾越长，不能和普通受击混淆。
- 敌人互撞时以接触点的金色火花和一次性震波标示伤害归因。
- 延迟炸弹有递进闪烁和可听的引信节拍，后续命中引爆时声音要明显更重。
- 殉爆以较弱的二次色调呈现，避免压过第一爆的阅读重点。

## 溅射物/区域/实体规则

爆炸是瞬时圆形冲击，不留下长期伤害区域。震荡波使范围内敌人从爆心向外击飞；最近敌人获得最明显的方向性，边缘敌人只受到较弱推力。被击飞敌人撞到另一敌人时产生一次碰撞伤害，并可把部分冲量传递出去；单次击飞链需要有限传递次数，避免在密集人群中无限弹跳。

火药压机每 6/5/4 次命中贴上一枚延迟炸弹，均为待试玩锚点；同一敌人基础最多带 1 枚，后续命中引爆后消失。殉爆只由爆炸击杀或撞墙触发，二次爆炸强度暂以原爆炸的 40/55/70% 为锚点，并限制连锁深度；场上同时待引爆的视觉标记也应有上限，防止玩家无法辨认。

## 反馈与可读性

玩家需要知道爆炸造成了什么：初爆伤害、击飞、敌人互撞、撞墙、二次爆炸应按时间分层显示。延迟炸弹必须可见、可计数并能区分“已贴附”和“将被引爆”。当敌人即将撞墙或撞入人群时，方向拖线帮助玩家把结果归因给自己的击发角度，而不是随机物理。

## 平衡边界

爆破强在改写敌群位置，不能在单体、群伤、控制和连锁上同时无条件领先。击飞冲量暂以 1.5/2/2.5 倍为试玩锚点，但必须有速度与链长上限。殉爆的连锁深度建议从 3 起试，觉醒后最多 5；二次爆炸不得完整继承一切二次触发。延迟炸弹应提供可安排的节奏，不应让每次命中都变成免费爆炸。

## 与其他流派的桥梁

燃烧弹 `incendiary` 让爆炸散布燃料，为火提供更快的铺设面。毒气弹 `gas_shell` 让爆炸释放短寿命毒雾，把爆炸覆盖转化为感染接种。冻脆 `brittle` 让冻结敌人受到爆炸或击飞时额外碎裂。爆破与闪电的协同是涌现式的：击飞和聚团后，带电敌人自然更容易相互放电，无需额外桥梁。

## 待确认问题

1. 玩家是否需要在引爆前看到最主要的击飞方向与可能的撞墙结果？
2. 延迟炸弹由任何后续命中引爆，还是只由弹珠命中引爆，哪种节奏更清晰？
3. 敌人碰撞的冲量传递需要多强，才能有保龄球感而不失控？
4. 殉爆的最大深度、衰减和视觉层级如何设定，才能既爽快又可读？
