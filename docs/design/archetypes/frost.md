# 冰封流派：物理改造战场

> 本文是玩法/美术规格草案，不是实现说明。文中全部数值均为待试玩锚点，须以实际对局反馈校准。

## 定位与幻想

冰封将敌人从“被攻击者”转化为可摆布的台面棋子。冻结后的敌人是可推动的冰球、暂时的障碍与可被击碎的风险源；玩家用弹珠轨迹和敌群站位制造碰撞、堵路、碎裂与碎片齐射。

## 印象与关键词

冷硬、停滞、折射、棋子、撞击、碎裂、临时墙、滑行。冰应让台面变得更具空间感，而非只是给敌人挂一层蓝色减速。

## 核心循环

冰霜弹珠累积霜冻并冻结目标；永冻将其保留为更大、更稳定的冰球障碍；玩家把弹珠或其他被位移的敌人撞向冰球，改变战场线路并让冰球高速撞击；冰爆在冰球（或普通冻结目标）以足够速度撞上敌人或墙面时触发——冰球碎裂、解除冻结，并沿运动方向射出定向冰碎片，完成一次以失去冰球为代价的破局。

## 弹珠与遗物协同

| 角色 | 名称 / id | 玩家可感知作用 |
|---|---|---|
| 起点 | 冰霜弹珠 / `blue_marble` | 累积霜冻并使敌人冻结。 |
| 触发 | 碎冰锤 / `ice_hammer` | 我方弹珠 Head 撞冻结敌人时造成局部碎裂（近身回报）。 |
| 台面 | 永冻 / `permafrost` | 冻结敌人化为更大、本发内不解冻的可推动冰球障碍，冰球撞击敌人/墙面造成方向性碰撞伤害。 |
| 兑现 | 冰爆 / `cryoclasm` | 冻结目标/冰球高速碰撞时碎裂，射出冰碎片抛射物（目标仍存活）。 |
| 桥梁 | 黑冰 / `black_ice`、冻脆 / `brittle` | 分别连接回响与炸弹（延后件，见「已决策」）。 |

**本轮拍板范围：** 核心原型为永冻 + 冰爆，与冰霜弹珠组成雏形；碎冰锤为现有件。黑冰与冻脆身份确认但延后至回响 / 爆破基础设施就绪后再补，不与核心原型一起承诺上线。

雏形是冰霜弹珠加永冻：先证明”做棋子”有乐趣；冰爆提供拆棋与远程收束，在冰球高速碰撞时碎裂释放碎片；碎冰锤奖励弹珠 Head 直接撞击冻结目标的近身碎裂。两件触发分工清楚：**碎冰锤 = 我方球撞冻结敌人；冰爆 = 冻结敌人/冰球自身高速撞墙或撞敌**，不抢同一次命中。冰爆不依赖永冻也能工作（普通冻结目标高速碰撞同样碎裂），永冻只提高冰球的可控性与命中机会。冰不能依靠持续伤害成为纯粹的安全流。

## 视觉语言

主色为深靛蓝、冰青、白色折光与极少量灰蓝阴影。冻结不是简单的半透明滤镜：敌人内部应有放射状裂纹和被封住的动作残像；永冻后的冰球体积更大、轮廓更圆，边缘带霜刺。冰碎片呈锐利三角形折射，黑冰则是接近墨色、带蓝紫反光的薄滑面。

## 图标方向

冰霜弹珠为内部有雪花裂纹的蓝白圆珠；碎冰锤为敲击冰块的锤面；永冻为包住敌人轮廓的巨大冰球；冰爆为向外放射的六枚冰片；黑冰为黑蓝镜面上的反光弧线；冻脆为覆盖霜纹的裂壳炸弹。图标可使用尖锐晶体轮廓，但保持中心物体清晰。

## 特效需求

- 霜冻积累以逐层加密的晶霜环表现，冻结瞬间有一次短而干净的结晶收束。
- 冰球被推动时显示低速滑痕、重量感和碰撞前的影子偏移。
- 碰撞优先呈现接触点的白亮裂纹，再呈现扩散碎屑，避免大面积蓝雾。
- 冰爆先让冻结外壳碎开，再射出可追踪的冰片；目标仍存活这一信息必须清楚。
- 黑冰反弹应带细长冷色擦痕和不同于普通墙面的滑音。

## 溅射物/区域/实体规则

冻结敌人保持为场上可见实体：基础冻结可恢复行动；拥有永冻时，冻结体积放大且至少维持到本发结束，成为可推动的冰球障碍。冰球与敌人或墙面碰撞时产生可见的冲击与碎裂伤害；冰球不应无限加速，也不能在一个区域反复无代价来回结算。

冰爆为**物理触发**：被冻结的敌人（含永冻冰球）以足够速度撞上敌人或墙面时碎裂（首轮速度阈值待试玩校准，约为冰球最大滑行速度的 50%），解除其冻结并清空其霜冻，但该目标仍然存活、不绕过生命。碎片数量只按遗物等级为 3/4/6 枚（首轮锚点），不读取隐藏霜冻层数；碎片自碎裂点沿冰球运动方向的扇区散开，优先选择该方向上不同的可见敌人，命中一次后消失，可击退但不连续分裂。觉醒后碎裂目标恢复时保留 2 层 Frost、碎片命中可再施加 Frost。冰爆不依赖永冻即可触发。黑冰是短寿命墙面薄冰，只由冰霜弹珠撞墙留下：按墙段存在，每段只保留最新一块，本发结束时清除，同场段数有上限。

## 反馈与可读性

冻结、可推动冰球、即将碎裂三者必须不同：普通冻结偏静止，永冻冰球有更粗的外壳与地面投影，冰爆选中的目标有短促的裂纹聚焦。冰碎片的数量、路径和命中顺序应可跟踪；碰撞伤害要有方向性，帮助玩家理解是自己摆出的冰球位置造成了收益。

## 平衡边界

冰应奖励空间操作，不能用永久控制压制敌人。冻结必须有可预期的恢复或本发结束边界；永冻的同场冰球数量需要上限，基础 2 个、觉醒 3 个。冰爆以失去一个冰球和该目标全部霜冻交换碎片收益，不能绕过目标生命。碎冰锤与冰爆触发分离：碎冰锤只在我方弹珠 Head 撞冻结目标时触发，冰爆只在冻结目标自身高速碰撞时触发，两者不抢同一命中。冰爆单独掉落时也必须能工作（普通冻结目标撞墙即可碎裂），永冻只提升可控性而非冰爆的开关。黑冰只保留速度或回响收益，不应同时提供大额伤害、无限反弹与永久墙面。

## 与其他流派的桥梁

热冲击 `thermal_shock` 使燃烧和冻结共存的目标碎裂并解除两态，连接火。黑冰 `black_ice` 将冰霜弹珠的撞墙转为自铺冰面，使回响弹珠在其上反弹不掉速。冻脆 `brittle` 让冻结敌人受到爆炸或击飞时额外碎裂，连接炸弹。冰与闪电的聚团收益属于自然涌现，不额外承诺专属桥梁。

## 已决策（首轮锚点，须以试玩校准）

1. **冰球推行手感**：冰球质量大、摩擦低，被撞后沿冲量方向直线滑行、**不主动转向**（它是被摆布的棋子，不是制导弹）。最大滑行速度约为弹珠 Head 上限的 40–50%（Head 800 → 约 320–400），线性阻尼使其约 1.5–2 秒停下，每次碰撞损失约 30% 动量，避免同区域无代价来回结算。
2. **永冻冰球上限**：按**全场**约束，基础同场最多 2 个、觉醒 3 个；达到上限时最早的冰球提前碎裂/解冻，而非禁止新冻结。
3. **冰爆碎片选敌**：以冰球运动/碰撞方向的扇区优先选择不同可见敌人，而非纯自动索敌；方向给玩家间接选择权，碎片命中即消失、不分裂。
4. **黑冰可见范围与寿命**：按墙段存在、每段只保留最新一块，本发结束清除；单块寿命短（约 2–3 秒），同场最多 3–4 段，消失前短促淡出预告，支持回响反弹但不让台面长期被蓝黑区域覆盖。

**等级差异锚点（与主设计文档 §5 对齐）：**

- 永冻：所有等级均提供"冻结体积放大 + 本发内不解冻 + 方向性碰撞伤害"；Lv1/2/3 碰撞伤害成长（首轮锚点 4/7/11），同场上限恒为 2；觉醒上限 +1（=3）且冰球碰撞时给被撞敌人施加 1 层 Frost。
- 冰爆：Lv1/2/3 碎片数 3/4/6；觉醒后碎裂目标恢复时保留 2 层 Frost、碎片命中可再施加 Frost。碎片数只按等级，不读取隐藏霜冻层数。
