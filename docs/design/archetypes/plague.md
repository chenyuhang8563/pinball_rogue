# 瘟疫流派：延迟宿主经营

> 本文是玩法/美术规格草案，不是实现说明。文中全部数值均为待试玩锚点，须以实际对局反馈校准。

## 定位与幻想

瘟疫不是把层数换成一次大爆炸，而是把敌人逐步培养成会移动、会扩散的传染媒介。玩家播种、等待成熟、改变敌群队形，再在宿主死亡时收获孢子区或孢子体形成的长期战场价值。

## 印象与关键词

潜伏、孵化、病变、寄生、迁徙、扩散、腐蚀、收割。它应令人感到“不安地增长”：开始安静，之后逐步让敌人群自己变成问题。

## 核心循环

毒液弹珠接种感染；感染按潜伏、发病、宿主三阶段成熟。潜伏期缩短成熟过程，发病带来减速，寄生让宿主向其他敌人靠拢。宿主死亡后，由瘟疫培养皿、脓爆提供的传播，把已成熟的威胁重新种回敌群，形成滚动但可控的感染生态。

## 弹珠与遗物协同

| 角色 | 名称 / id | 玩家可感知作用 |
|---|---|---|
| 起点 | 毒液弹珠 / `green_marble` | 命中接种潜伏感染。 |
| 传播 | 瘟疫培养皿 / `poison_culture` | 宿主死亡时传染邻近敌人，并保留成熟价值。 |
| 成熟 | 潜伏期 / `incubation` | 加快阶段推进，使发病减速更早出现。 |
| 聚集 | 寄生 / `parasite` | 宿主主动靠向其他敌人，成为移动传染源。 |
| 兑现 | 脓爆 / `pustule` | 宿主死亡后留下孢子区或孢子体继续传播。 |
| 桥梁 | 腐蚀 / `corrosion`、瘴气 / `miasma` | 分别连接物理/回响与火。 |

雏形应为毒液弹珠与潜伏期或培养皿；脓爆是将“等成熟”转成地图控制的兑现件；寄生改变队形，价值应高于单纯加伤。任何击杀都可以收割已成熟宿主，但不能以技能作为独立构筑轴。

## 视觉语言

主色为病绿、灰黄、深青和少量湿润的乳白高光。潜伏阶段只有皮肤下缓慢游动的细线；发病阶段出现不规则斑块与滴落；宿主阶段形成较明显的孢囊、菌丝拖尾和朝敌群偏移的行动感。整体避免火焰式高亮爆闪，强调黏连、呼吸与缓慢扩张。

## 图标方向

毒液弹珠为装有浑浊绿液的透明珠；培养皿为裂开的圆皿与菌落；潜伏期为带刻度的蛹状孢囊；寄生为带触须的心脏或虫体；脓爆为破裂孢囊外放孢子；腐蚀为被酸液蚀穿的盾牌；瘴气为绿色云团中夹一簇橙火。图标轮廓应偏圆、偏生物，避免与冰晶或闪电混淆。

## 特效需求

- 阶段转换要有明确、低频的脉冲，而非持续噪点。
- 发病减速以拖影变长、动作节奏变慢表现，不依赖文字提示。
- 宿主靠拢时以细菌丝或孢尘流指出其移动意图，不能像随机寻路。
- 孢子区边缘应有缓慢收缩的菌丝环，中心密度更高，消散时从外向内枯萎。
- 孢子体需要可辨认的移动轮廓和明确落点，音效可用湿润破裂与低沉呼吸感。

## 溅射物/区域/实体规则

感染有三阶段：潜伏仅造成轻微存在感；发病使目标减速；宿主会朝附近其他敌人靠近。阶段进度与阶段本身都应可见，宿主不能在场上无期限存在：死亡、离场或本发结束都会终止其宿主行为。

脓爆在宿主死亡时二选一产生传播媒介：孢子区固定在死亡位置，持续短时间接种进入范围的敌人；孢子体则选择最近的未感染敌人爬行，抵达后接种并消失。孢子区半径暂以 100/120/140 为锚点；孢子体基础数量上限为 1，觉醒后可加 1。单个敌人不应被同一批孢子体反复刷新为无限宿主。

## 反馈与可读性

感染状态要让玩家不用读数也能区分三阶段：潜伏细线、发病斑块、宿主孢囊。宿主将要传播时，附近可被影响敌人出现短暂的虚线连结；孢子区的有效边界始终清晰。收割发生时要先显示宿主破裂，再显示区域或孢子体诞生，避免玩家误解为纯伤害爆炸。

## 平衡边界

毒的收益必须以等待、目标选择和敌群布局为代价，不能通过叠层伤害抢走火的爆发身份。阶段数固定为三段；成熟速度、减速幅度和吸引距离均为待试玩锚点。宿主吸引半径暂以 80/110/140 为锚点，但不应使敌人完全失去行动能力。孢子区必须有寿命与场上数量上限，建议基础同时存在 1 个，以防整个台面长期无成本覆盖。

## 与其他流派的桥梁

腐蚀 `corrosion` 让感染阶段越深的敌人护甲越低，为物理与回响流派制造脆弱目标，但不直接替其结算伤害。瘴气 `miasma` 让发病或更深阶段的感染目标遇火爆燃，消耗感染换来一次毒火爆发。炸弹的毒气弹 `gas_shell` 适合把爆炸范围转化为广域接种。毒与闪电、毒与冰不设专属遗物桥梁，保留普通共存与经炸弹间接衔接的空间。

## 待确认问题

1. 脓爆应默认生成孢子区，还是在敌群稀疏时优先生成孢子体？
2. 宿主靠向敌群的行为是否足够可控，是否需要限定可选目标的最近距离？
3. 孢子区的存续时间与场上上限怎样设定，才能有经营感而不堵塞读图？
4. 发病阶段的减速应以多大幅度提供战略价值而不替代冰的控制身份？
