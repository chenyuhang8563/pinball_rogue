# 回响流派：球体进化

> 本文是玩法/美术规格草案，不是实现说明。文中全部数值均为待试玩锚点，须以实际对局反馈校准。

## 定位与幻想

回响把每一次漂亮反弹存进弹珠本身。玩家不是只追求立刻命中，而是通过挡板与墙面的操作让球不断充能，随后进入穿透、留痕或巨大化的质变状态；越长、越干净的轨迹，越能换来有分量的终结。

## 印象与关键词

共振、回弹、蓄势、轨迹、穿透、余像、巨大化、终结。它应像一颗越弹越有生命的球，带着玩家亲手维持的节奏感。

## 核心循环

大地弹珠从非敌人碰撞中获得回响；共振核心在达到档位后使本发进入共振态并穿透敌人；残影把这段运动变成短暂的伤害轨迹；雪崩把累积的碰撞和轨迹长度结算成一次巨大化撞击。核心决策是“继续保球充能”还是“用当前状态穿过敌群兑现”。

## 弹珠与遗物协同

| 角色 | 名称 / id | 玩家可感知作用 |
|---|---|---|
| 起点 | 大地弹珠 / `brown_marble` | 反弹积累回响，命中附带回响收益。 |
| 变形 | 共振核心 / `resonance_core` | 满档后让球进入穿透性的共振态。 |
| 延展 | 残影 / `afterimage` | 共振态的行进路线留下伤害轨迹。 |
| 兑现 | 雪崩 / `avalanche` | 本发碰撞达到阈值后巨大化并重击。 |
| 桥梁 | 带电球 / `live_wire` | 共振反弹产生放电，连接闪电。 |

雏形为大地弹珠和共振核心；残影把操作成果转成空间控制；雪崩为高技术玩家提供一发一度的终结。冲刺等辅助只应帮助修正轨迹或保住充能，不能直接制造回响层数。

## 视觉语言

主色为土褐、琥珀金、深紫影与明亮的共振白线。普通回响以同心细环环绕弹珠；档位越高，环越稳定而不是越杂乱。共振态的球面可有轻微拉长的折射和多层余像；残影是半透明的暖色弧带；雪崩巨大化像一颗被压力撑开的矿石球，边缘出现厚重尘环。

## 图标方向

大地弹珠为带矿物纹理的褐色圆珠；共振核心为中心圆核外套三层同心环；残影为一颗球拖出双重弧线；雪崩为巨大石球压向小敌影；带电球为环绕微小电弧的圆球。图标需突出圆形和轨迹，不使用冰晶或爆炸云来代替共振语言。

## 特效需求

- 每次有效反弹增加一圈可数、可听的共振反馈，音高逐档上行。
- 达到共振态时，环由断续变为完整，球的尾迹立刻拉长。
- 穿透敌人要显示球体穿过而非把敌人弹飞，命中点留下短暂的圆形波纹。
- 残影持续时间短、边缘柔和，但有效路径边界清晰。
- 雪崩先有球体膨胀和低频蓄势，再出现大范围撞击与尘环，不能突兀放大。

## 溅射物/区域/实体规则

共振态是本发内的球体状态：球能连续穿过敌人，但同一敌人一次穿越只获得一次主要命中，避免在重叠区域多次重复结算。残影为沿球最近轨迹铺出的短寿命带状区域；每条轨迹有固定持续时间，区域过多时优先淡出最早的一条，以保证台面可读。

雪崩每发最多触发一次。达到待试玩阈值 15/12/10 次有效碰撞后，球短暂巨大化；它在这段期间命中敌人产生一次范围重击和击退，之后恢复正常。巨大化不增加无限存续时间，也不应借墙面无限续延。带电球的放电每次反弹只寻找一个最近可见敌人，单次电弧结束即消失。

## 反馈与可读性

回响需要明确展示“现在有几档、离共振差多少、这一发是否已用过雪崩”。共振环、球尾长度和音阶构成三重提示；残影必须区分于毒的圆形孢子区和火的冲击波。雪崩可触发时，弹珠边缘出现短时加粗的脉冲；触发后立即熄灭，防止玩家误以为可以重复巨大化。

## 平衡边界

回响收益来自保球和操作质量，不能由普通命中快速充满。共振所需反弹数暂以 8/6/5 为锚点，雪崩阈值暂以 15/12/10 为锚点，均需按实际一发时长调整。穿透应提高路线价值而非无限堆叠单体伤害；残影需数量或总长度上限；巨大化一发一次并有软上限，避免长轨迹必然清屏。

## 与其他流派的桥梁

黑冰 `black_ice` 由冰霜弹珠在墙上铺设，使回响球反弹时保留更多速度并可获得额外充能。带电球 `live_wire` 让共振态反弹摩擦起电，向最近敌人放电并可为闪电体系提供带电目标。腐蚀 `corrosion` 让深度感染目标更适合被回响的物理命中处理。回响与炸弹可通过空间碰撞自然互补，但不强设专属桥梁。

## 待确认问题

1. 回响档位应显式显示为数字、环数，还是两者兼用？
2. 共振态是否需要在首次穿透后短暂停顿，以强化质变瞬间的手感？
3. 残影更适合固定宽度还是随共振档位增长，怎样不会遮挡台面？
4. 雪崩的巨大化是否应优先服务范围撞击，还是更强调一次可控的击退方向？
